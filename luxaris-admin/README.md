# Luxaris Admin Panel

**Root User Only** - Administrative control center for the Luxaris platform.

---

## Overview

The Luxaris Admin Panel is a specialized interface exclusively for root users (system administrators) to manage and monitor the entire platform. This is where platform-wide control and observability converge.

## Key Capabilities

### User Management
- **Approve or reject** new user registrations
- Review pending user accounts
- Manage user roles and permissions
- Monitor user activity across the platform

### Platform Observability
- **Request Analytics** - View all HTTP requests with timing, status codes, and performance metrics
- **System Logs** - Track technical errors, warnings, and critical issues in real-time
- **System Events** - Monitor business events (user actions, post creations, publishing)
- **Audit Logs** - Security and compliance tracking for all platform activities

### System Health & Statistics
- API performance metrics and response times
- Error rates by endpoint
- User growth and activity trends
- Publishing success/failure rates
- Platform-wide health status

### Global Controls
- Feature flag management
- System configuration
- Emergency controls and maintenance mode
- Platform-wide announcements

---

## Access

This panel is **restricted to root users only**. The first registered user automatically becomes a root user with full administrative access.
